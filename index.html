Tentang Saya & Website Ini

Saya adalah lulusan SMK jurusan Rekayasa Perangkat Lunak (RPL) yang baru saja melanjutkan studi di bidang Informatika. Dengan pengalaman dalam berbagai bahasa pemrograman selama di SMK, saya membangun website ini sebagai portofolio sekaligus menawarkan jasa pembuatan website.

Website ini bertujuan untuk memperkenalkan keterampilan saya dalam pengembangan web serta membantu mahasiswa, profesional, dan pemilik bisnis kecil yang membutuhkan website berkualitas dengan harga terjangkau.

Dengan keahlian yang saya miliki, saya siap membantu Anda mewujudkan website impian, baik untuk keperluan tugas kuliah, profil profesional, maupun bisnis. Silakan jelajahi portofolio saya dan jangan ragu untuk menghubungi saya untuk diskusi lebih lanjut.
